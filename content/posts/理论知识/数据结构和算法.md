# 查找 
评价指标:
- 查找长度(SL)
- 平均查找长度(ASL):考虑查找成功和失败的平均SL

##### 顺序查找 O(n)
```python
def search_seq(ls,k):
	for i in range(len(ls)):
		if k == ls[i]:
			return i
	return -1
```

##### 二分查找 O(Log2n)
```python
def search_bin(ls,k):
	low = 0
	high = len(ls)-1
	while low<=high:
		mid = (high-low)/2
		if ls[mid] == k:
			return mid
		elif ls[mid]<k:
			high = mid+1
		else:
			low = mid-1
```
如果取整为向上(向下),则左子树比右子数量大(小)1个或相等
